# Agenda de Turnos

Sistema web de agenda de turnos que permite registrar, visualizar y eliminar horarios de trabajo de manera sencilla. La aplicación facilita la organización del personal mediante una interfaz intuitiva donde se ingresan datos como nombre del empleado, tipo de turno, horario y fecha.

El sistema almacena la información localmente en el navegador, lo que permite mantener los datos incluso después de cerrar la página, sin necesidad de conexión a internet ni base de datos externa.

Este proyecto fue desarrollado utilizando tecnologías web básicas como HTML, CSS y JavaScript, enfocándose en la usabilidad, simplicidad y funcionalidad.
